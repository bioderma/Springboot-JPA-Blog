# Springboot-JPA-Blog

# 프로젝트 개요
- **프로젝트명** : Springboot-JPA-Blog
- **프로젝트 소개** : 온라인 상에서 통상적으로 서비스되는 블로그의 게시판내용을 구현
- **프로젝트 주제 선별 이유** : JAVA 개발자 과정을 이수하면서 그동안 배웠던 것들을 활용하고자 주제로 선별하였음.
- **개발 인원** : 1명.
- **개발 기간** : 20일.

- **주요 기능** 
  - `Spring Security`를 활용한 로그인 / 카카오 로그인구현
  - 게시글의 `댓글` 작성 기능.
  - 게시글 작성 시 `이미지` 첨부 기능.
  - `회원정보 확인 및 변경` 기능.
  
- **백엔드 개발 언어** 
  - `Java`
  
- **백엔드 개발 환경** 
  - `Windows 10`
  - `Eclipse`
  - `SpringBoot`
  - `Maven`
  - `JPA`
  - `Spring Security`
  
- **프론트 개발 환경 및 언어**
  - `HTML5`
  - `ThymeLeaf`
  - `Bootstrap`
  - `JavaScript`
  
- **데이터베이스**
  - `MariaDB`
  
- **관리**
  - `Github`
  
# 프로젝트 상세



## 회원 기능
-기본로그인과 카카오로그인 두가지로 진행이가능함

  
## 마이페이지 기능
- 사용자가 가입한 계정의 정보를 조회 및 변경할 수 있음.
